ThystTV offers extensive customization options to make the app truly yours. This guide covers the main ways to personalize your viewing experience.

## Theme Customization

### Choosing Your Theme

ThystTV supports multiple theme options:

<Accordion title="Light & Dark Themes">

**Basic Theming**
- Light theme for daytime viewing
- Dark theme for nighttime or low-light environments
- Auto-follow system theme

**Advanced Theme Scheduling**

You can set different themes for when your system is in light vs. dark mode:

1. Enable "Follow system theme"
2. Set "Dark theme on" - theme used when system is in dark mode
3. Set "Dark theme off" - theme used when system is in light mode

This allows you to use one theme during the day and another at night automatically.

</Accordion>

### Visual Style

<Accordion title="Material Design Options">

**Rounded Corners**
- None - Sharp, rectangular corners
- Small - Subtle rounding
- Medium - Moderate rounding (default)
- Large - Pronounced rounding

**Layout Density**
- **Reduced padding**: More content visible on screen
- **Compact text**: Tighter text spacing
- Combine both for maximum information density

**Interface Elements**
- **App bar lift**: Elevation effect when scrolling
- **Bottom navigation color**: Accent color on bottom bar
- **Material 3**: Google's latest design system

</Accordion>

<Note>
Material 3 provides improved color theming and more modern visual styling.
</Note>

## Chat Overlay Customization

### Floating Chat

One of ThystTV's signature features is the floating chat overlay:

<Accordion title="Setting Up Floating Chat">

**Enable Floating Chat**
1. Go to Settings → Chat Settings
2. Enable "Floating chat enabled"
3. While watching a stream, double-tap to cycle chat modes

**Chat Modes**
- **Overlay**: Chat floats over the video
- **Side-by-side**: Traditional split view
- **Hidden**: Video only

**Customization Options**
- **Transparency**: Adjust from 0-100% (Settings → Floating chat transparency)
- **Position**: Drag the chat window anywhere on screen
- **Size**: Resize by dragging the corner handles
- **High visibility mode**: Adds background tint for better readability

</Accordion>

<Info>
Your floating chat position and size are remembered between sessions.
</Info>

### Chat Appearance

<Accordion title="Message Styling">

**Username Display**
- **Bold usernames**: Makes usernames stand out
- **Random colors**: Assigns colors to users without custom colors
- **Theme-adapted colors**: Adjusts colors for better contrast

**Message Elements**
- **Timestamps**: Show time each message was sent
- **Timestamp format**: 12-hour or 24-hour
- **Badges**: Show subscriber, moderator, and third-party badges

**Size Adjustments**
```
Text size: 14 (default)
Emote size: 29.5 (default)
Badge size: 18.5 (default)
Chat size modifier: 100% (default)
```

Adjust these to make chat more readable on your device.

</Accordion>

<Accordion title="Emote Customization">

**Third-Party Emotes**

Enable emotes from popular services:
- **7TV (SevenTV)**: Modern emote platform with live updates
- **BTTV (BetterTTV)**: Classic third-party emotes
- **FFZ (FrankerFaceZ)**: Another popular emote provider

**7TV Advanced Features**
- **Personal emotes**: See emotes specific to individual users
- **Name paints**: Custom username color effects
- **7TV badges**: Custom badges from 7TV
- **Live updates**: Emotes update in real-time

**Emote Behavior**
- **Animated emotes**: Enable/disable GIF animations
- **Zero-width emotes**: Overlay emotes (for emote combos)
- **System message emotes**: Show emotes in system messages

**Image Quality**

Balance quality vs. data usage:
- **1x**: Lowest quality, minimal data
- **2x**: Medium quality
- **3x**: High quality
- **4x**: Maximum quality (default)

Enable **WebP support** for better compression with same quality.

</Accordion>

## Gesture Controls

### Configuring Gestures

<Accordion title="Gesture Zones">

ThystTV uses a zone-based gesture system:

**Vertical Gestures**
- **Left half**: Brightness control
- **Right half**: Volume control

**Horizontal Gestures (VODs only)**
- **Top half**: Seek through video
- **Bottom half**: Playback speed adjustment

**Zone Split**

Adjust the vertical split point between top and bottom zones in Settings → Player Settings → Gesture zone split.

</Accordion>

<Accordion title="Sensitivity & Feedback">

**Sensitivity Levels**
- **Low**: Requires larger swipes
- **Normal**: Balanced (default)
- **High**: Very responsive

Find your preference in Settings → Player Settings → Gestures sensitivity.

**Haptic Feedback**

Enable vibration feedback during gesture adjustments:
- Provides tactile confirmation
- Disabled by default to save battery
- Toggle in Settings → Player Settings → Gestures haptic

</Accordion>

<Info>
Gestures are automatically disabled when player controls are visible to prevent accidental adjustments.
</Info>

## UI Layout

### Grid Layout

<Accordion title="Content Grid Customization">

**Column Count**

Adjust how many columns display in different orientations:

Settings → UI Settings → Portrait column count
Settings → UI Settings → Landscape column count

More columns = more content visible, but smaller previews.

**Stream Previews**

Control thumbnail visibility:
- **Disabled**: Always show previews
- **Always**: Hide all previews (compact mode)
- **Wi-Fi only**: Hide on cellular to save data

</Accordion>

### Navigation Tabs

<Accordion title="Tab Organization">

Customize which tabs appear and their order:

**Main Navigation**
- Reorder bottom navigation tabs
- Hide tabs you don't use
- Settings → UI Settings → Navigation tabs

**Section Tabs**
- Following section tabs
- Saved section tabs
- Channel page tabs
- Game page tabs
- Search tabs

Each can be customized independently.

**Start Screen**

Choose which Following tab opens by default:
- Followed streams
- Followed channels
- Followed games

</Accordion>

## Quality Presets

### Stream Quality

<Accordion title="Quality Configuration">

**Default Quality**

Set your preferred stream quality:
- Auto (adapts to connection)
- 1080p60
- 720p60
- 720p
- 480p
- 360p
- 160p
- Audio Only

**Data Saver Mode**

When enabled, automatically uses lower quality on cellular:

1. Enable "Data saver" in Player Settings
2. Set "Default cellular quality"
3. Wi-Fi uses standard quality, cellular uses your preset

</Accordion>

<Note>
Quality setting "Saved" remembers your last manual selection per stream.
</Note>

## Chat Translation

### On-Device Translation

<Accordion title="Setting Up Translation">

**Enable Translation**

1. Go to Settings → Chat Settings
2. Enable "Translate chat messages"
3. Set "Translate target language" (e.g., English)
4. Download required language models

**Translation Modes**

- **Manual**: Tap a message to translate it
- **Automatic**: Translate all messages in real-time

**Language Models**

Manage downloaded languages in Settings → Chat Settings → Downloaded languages

- Models are stored on-device
- No internet required after download
- Delete unused models to save space

</Accordion>

<Info>
Translation is powered by on-device ML models - your chat data never leaves your device.
</Info>

## Advanced Customization

### Proxy & Network

<Warning>
Advanced settings - modify only if you understand the implications.
</Warning>

<Accordion title="Stream Proxy">

**Custom Proxy URL**

Replace Twitch's default stream delivery:

- Enables alternative backends
- Can bypass restrictions
- Allows playlist manipulation

Default: `https://api.ttv.lol/playlist/$channel.m3u8`

**Custom Headers**

Add custom HTTP headers to stream requests for advanced use cases.

</Accordion>

### Buffer Settings

<Accordion title="Playback Buffer Tuning">

Adjust buffering for your network:

**Higher Values** (better for unstable connections)
- More buffering
- Less stuttering
- Higher latency

**Lower Values** (better for stable connections)
- Less latency
- More responsive
- Risk of stuttering

**Settings**
- Min buffer: 15000ms (default)
- Max buffer: 50000ms (default)
- Buffer for playback: 2000ms (default)
- Rebuffer: 2000ms (default)

</Accordion>

## Personalization Tips

<Accordion title="Recommended Configurations">

**For Mobile Data Users**
- Enable Data Saver
- Set cellular quality to 480p or lower
- Hide stream previews on cellular
- Use compact number formatting

**For Tablet Users**
- Increase portrait/landscape columns
- Enable "Move player freely"
- Adjust chat width for better split view
- Use larger emote/text sizes

**For Chat-Focused Viewers**
- Enable floating chat
- Adjust transparency for your preference
- Enable all third-party emote providers
- Increase message limit
- Enable 7TV live updates

**For Minimal UI**
- Enable reduced padding
- Enable compact text
- Hide unused player menu buttons
- Reduce navigation tabs
- Hide stream uptime/tags

</Accordion>

## Saving Your Configuration

### Backup & Restore

<Accordion title="Preserving Your Settings">

**Backup Settings**

1. Go to Settings → Backup settings
2. Choose save location
3. File includes all preferences and customizations

**Restore Settings**

1. Go to Settings → Restore settings
2. Select backup file
3. All settings will be restored

<Note>
Backup before updating the app or switching devices.
</Note>

</Accordion>
